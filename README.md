## **1. 프로젝트 개요**

### 1.1 프로젝트 요약
- **핵심 아이디어**:  
  - 스케치를 기반으로 **가구 이미지** 또는 **3D 모델**을 자동 생성  
  - 생성된 결과와 유사한 가구를 **추천**해주는 웹 서비스
- **기대 효과**:
  - **디자인 프로세스** 효율화 (시각화가 빠르고 직관적)
  - 사용자의 아이디어를 즉시 **시각화**하며, 유사 가구 추천으로 **사용자 경험** 향상
